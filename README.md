### About / Giới thiệu

**EN** &nbsp; I'm a mid-level **AI/LLM Engineer** specializing in **retrieval-augmented generation**. I take RAG systems from prototype to production: ingestion, chunking, retrieval, evaluation, and a reliable API. I care about eval and guardrails as much as the demo, and I favor small, well-tested systems over heavy frameworks.

**VI** &nbsp; Mình là **AI/LLM Engineer** (mid-level), chuyên về **RAG**. Mình đưa hệ thống RAG từ prototype lên production: ingest dữ liệu, chunking, retrieval, đánh giá (eval) và API ổn định. Mình coi trọng eval và guardrails ngang với bản demo, ưu tiên hệ thống nhỏ gọn, được kiểm thử kỹ.

- Production mindset: deploy, monitor, iterate
